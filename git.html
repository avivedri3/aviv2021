Welcome to GIT WORLD ,

גיט מאפשר לכם כמו במשחק מחשב לשמור את המצב הנוכחי שלכם רק בתכנות

  • git init - מתקין גיט בתקיית פרוייקט כדי שנוכל לבצע פקודות של גיט
  • git add . - מכין את הקבצים שנרצה לשמור בענף/בראנץ
  • git commit -m "message you want to enter for documentation" - שומר את הקבצים שהכנסנו בפקודה של האד ובפעם הראשונה גם מייצר בראנץ' חדש בשם מאסטר/מיין שהוא הבראנץ הראשי
  • git branch - מציג את רשימת הבראנצים בפרוייקט, ירוק זה הנוכחי
  • git bracn {banch_name} - יצור ענף/בראנץ' חדש בשם שניתן לו
  • git checkout {branch_name} - יעבור לענף
  • git checkout -b {branch_name} - גם מייצר את הענף וגם עובר אליו
  • git log - מציג את כל המיילסטון / שינוי שמנים שקראו לאותו בראנץ וניתן גם לשחזר בואתו בראנץ את ההיסטוריה שלו
